Download and import the archive

Here is the complete archive from January 2015 until mid July 2026.

This archive is an .mbox file and needs to be imported into Mozilla Thunderbird to read it. For security reasons email addresses have been redacted.

Download the RoundTable archive MBOX file

  1. Install Thunderbird if you don't already have it.
  2. Install the "ImportExportTools NG" add-on, which adds support for importing MBOX files.
    • Open Thunderbird.
    • Press Alt to reach the menu bar.
    • Go to Tools > Add-ons and Themes.
    • Search for "ImportExportTools NG".
    • Install it and restart Thunderbird if prompted.
  3. In the Folder Pane, find "Local Folders". If you tab around you should find it.
  4. When you've found "Local Folders", press the Applications key and choose "New Folder", this is where you'll import the .mbox file into. Choose a name for the new folder and tab to "Create". Import the MBOX file.
    • Press the Applications key (or Shift+F10) on your newly created folder. If you can't find it, make sure you're in the "Local Folders" treeview.
    • Choose "ImportExportTools NG".
    • Choose "Import MBOX file".
    • Select "Import individual MBOX file."
    • Browse to your .mbox file and open it.

The messages should then appear in Thunderbird just like normal emails. Email me at taraowton@gmail.com or write to the RoundTable list at blindlanguageprofessionals@googlegroups.com if you have any problems.
